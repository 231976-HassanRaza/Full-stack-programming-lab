Accessibility

The web is for everyone — building accessibly is both ethical and smart.

✓
Semantic HTML

Use proper elements like <button>, <nav>, <main> so screen readers understand structure.

✓
Keyboard Navigation

Every interactive element must be reachable and operable via keyboard alone.

✓
Colour Contrast

Text must meet WCAG AA contrast ratios — at least 4.5:1 for normal text.

✓
ARIA Roles & Labels

Supplement HTML with ARIA only when native semantics aren't enough.

"Accessibility allows us to tap into everyone's potential."
— Debra Ruh
